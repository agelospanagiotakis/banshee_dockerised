This is a simple example for running a docker container with PHP-FPM and NGINX.

## Get it up and running

docker-compose build

docker-compose up -d 

docker-compose exec -it docker.apache sh
docker-compose exec -it docker.mysql bash

banshee http://127.0.0.1:8080/
banshee login http://127.0.0.1:8080/login

phpmyadmin http://127.0.0.1:8082/
